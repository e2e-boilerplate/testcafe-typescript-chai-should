# TestCafé Boilerplate

TestCafé end-to-end test automation boilerplate, Using TypeScript.

## Getting Started

    	1. git clone git@github.com:e2e-boilerplate/testcafe-typescript.git
    	2. cd testcafe-typescript
    	3. npm install
    	4. npm run test
